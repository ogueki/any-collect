# sprites — 妖精の表情差分イラスト置き場

`Sprite2DRenderer` はこのフォルダの PNG を自動で読み込みます（`src/lib/character/Sprite2DRenderer.tsx`）。
ファイルを置いて dev サーバを再読込すれば反映されます。画像が無い表情は `neutral.png` に、
`neutral.png` も無ければ絵文字プレースホルダーにフォールバックします。

## 必要なファイル（ファイル名 = 表情キー）
| ファイル名 | 表情 | 用途 |
|---|---|---|
| `neutral.png` | ふつう | 既定。最低これ1枚あればOK |
| `happy.png` | うれしい | 生成成功・レア獲得など |
| `surprised.png` | おどろき | 撮影・新発見 |
| `thinking.png` | かんがえ中 | 鑑定中・ローディング |
| `sad.png` | しょんぼり | 失敗・エラー時 |

## 推奨スペック
- **形式**: PNG、**背景は透過（アルファ）**
- **サイズ**: 1024×1024（正方形）。アプリ表示は最大 ~176px なので 512px でも可
- **構図**: 全カット**同じポーズ・大きさ・位置・画風**で、**顔の表情だけ**変える（差し替え時にガタつかない）
- **余白**: 見切れないよう周囲に余白を残す
